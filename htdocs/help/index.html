Type of user roles

  • Clients:They have only access to the frontend and to modify part of there own profile.
  • Readers:They don't use the system but they are registered to asign book reports to there profile.
  • Administrators:They have full access to the frontend and the backend.

To create system users you will need to login as an admin user and access to the backend.

Try to fill as much information as possible. If you don't upload a picture of the user the system will create a robot illustration unique for this name.

You will need to know the company the user bellongs in order to save the user. This is very important because the user will only see the books categories alloed for the company selected.

Books

Before you start to create a new report for a book you will need the following details:

  • Book author

    You can create the author directly when you create the report or you can go to: Authors

  • Contacts for books to populate the Rights area.
    • Publisher
    • Rights Owner
    • Primary Agent
    • US Agent
    • Translation Agent
    • TV Rights sold by
    • TV Rights bought by
    • Film Rights sold by
    • Film Rights bought by
    • Audio Rights sold by
    • Audio Rights bought by

    These contacts are not mandatory but you need to create them in oreder to be able to select them in the different dropbox.

    It is also possible to create them from the Create Book screen but mabye you will need to save progress to see the changes.

  • Content for the Report area

    It is very important that you clean of any format the content for the "Things you need to know" and "Main report content" areas.
    If you want to have some type of simple format as bold, italic or quotations you need to do this with in the system to be sure we have consistency in the design.
    The best whay to remove the format it is to pass the text to a plain no format text editor:
    Mac:
    TextEdit is the best and comes bundled with OSX. If you want to make it without any text formatting, go to menu “Format > Make Plain Text” (or shortcut 'Command+Shift+T”. You can also set this non-formatting as default everytime you launch this app.
    Windows:
    Notepad comes with windows and has not format at all.

    If you want to double check that the content has not format you can use the code button in the editor and see if it is some kind of extra formating like fonts or other unwanted tags.
